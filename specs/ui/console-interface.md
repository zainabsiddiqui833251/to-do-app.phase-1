# UI Specification: Console Interface

## Requirements
- Main loop with numbered menu:
  1. Add task
  2. List tasks
  3. Update task
  4. Delete task
  5. Mark complete/incomplete
  6. Exit
- Clear screen between operations (optional but nice).
- Input validation and helpful messages.
- Graceful exit on option 6.

## Tone
Friendly, clear, minimal clutter.
